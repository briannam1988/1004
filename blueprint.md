
# Blueprint: AI 상황별 문구 생성기

## 1. 프로젝트 개요

**AI 상황별 문구 생성기**는 사용자가 특정 상황(사과, 요청, 보고 등)과 원하는 톤, 그리고 구체적인 맥락을 입력하면, Google Gemini AI가 그에 맞는 자연스럽고 '초개인화'된 한국어 문구를 실시간으로 생성해주는 웹 애플리케이션입니다. 이 도구는 범용 AI 사용에 어려움을 느끼는 사용자를 위해, 특정 목적에 최적화된 쉽고 강력한 솔루션을 제공하는 것을 목표로 합니다.

---

## 2. 디자인 및 기능 명세 (현재 버전)

### 디자인
- **레이아웃:** 반응형, 모바일 우선, 단일 페이지, 중앙 정렬 컨테이너 구조
- **폰트:** Pretendard 웹 폰트 적용
- **색상:** 모던하고 활기찬 색상 팔레트 (Primary: SlateBlue)
- **시각 효과:** 카드, 버튼에 부드러운 그림자 효과 적용

### 기능
- **AI 기반 문구 생성:** Google Gemini AI를 연동하여 상황/톤/상세 맥락에 맞는 초개인화된 문구 실시간 생성
- **로딩 상태 표시:** AI 응답 대기 시간 동안 로딩 스피너 활성화
- **생성 기록:** `localStorage`를 사용해 최근 5개 문구 저장 및 조회
- **복사하기:** 생성된 문구를 클립보드에 복사
- **SEO 최적화:** 메타 태그, Open Graph, 구조화된 데이터(JSON-LD), robots.txt, sitemap.xml 적용
- **Google 소유권 확인:** Google Search Console 연동을 위한 인증 메타 태그 삽입

## 3. 현재 요청 계획: Cloudflare를 통한 배포

### 목표
GitHub와 연동된 Cloudflare Pages를 사용하여, 안정적이고 확장성 높은 글로벌 CDN 환경에 애플리케이션을 배포합니다. 이를 통해 코드 변경 시 자동으로 배포가 이루어지는 CI/CD(지속적 통합/지속적 배포) 파이프라인을 구축합니다.

### 세부 실행 계획

1.  **배포 환경 준비:**
    *   `package.json` 파일에 Cloudflare가 서버를 실행할 수 있도록 `start` 스크립트(`"start": "node server.js"`)를 추가합니다.
    *   `server.js` 파일에서 서버가 하드코딩된 포트(`3000`)가 아닌, Cloudflare 환경 변수(`process.env.PORT`)를 사용하도록 수정하여 유연성을 확보합니다.

2.  **사용자 가이드 제공:**
    *   **GitHub 리포지토리 생성:** 현재 프로젝트 코드를 올릴 새로운 GitHub 리포지토리를 만드는 방법을 안내합니다.
    *   **프로젝트와 GitHub 연동:** 로컬 프로젝트를 생성된 GitHub 리포지토리에 연결하고 코드를 푸시(push)하는 Git 명령어(`git init`, `git remote add`, `git add`, `git commit`, `git push`)를 안내합니다.
    *   **Cloudflare Pages 프로젝트 생성:**
        *   Cloudflare 대시보드에 로그인하여 Pages 메뉴로 이동하는 방법을 안내합니다.
        *   생성한 GitHub 리포지토리를 선택하고 연동하는 과정을 설명합니다.
        *   **빌드 설정**을 구성하는 방법을 안내합니다.
            *   **프레임워크 프리셋:** `None`
            *   **빌드 명령어:** (비워둠 - 우리는 빌드 과정이 없음)
            *   **출력 디렉터리:** (비워둠)
        *   **환경 변수** 설정: `GEMINI_API_KEY`를 Cloudflare 프로젝트 설정에 안전하게 추가하는 방법을 안내합니다.
    *   **배포 확인:** Cloudflare에서 제공하는 고유한 `*.pages.dev` URL을 통해 배포된 사이트를 확인하는 방법을 안내합니다.
