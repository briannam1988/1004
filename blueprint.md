
# Blueprint: AI 상황별 문구 생성기

## 1. 프로젝트 개요

**AI 상황별 문구 생성기**는 사용자가 특정 상황(사과, 요청, 보고 등)과 원하는 톤(정중, 부드럽게 등)을 선택하면, 그에 맞는 자연스러운 한국어 문구를 생성해주는 웹 애플리케이션입니다. 이 도구는 업무나 일상에서 메시지 작성을 어려워하는 사용자를 돕는 것을 목표로 합니다.

---

## 2. 디자인 및 기능 명세 (현재 버전)

### 디자인
- **레이아웃:** 단일 페이지, 중앙 정렬 컨테이너 구조
- **색상:** 흑백 기반의 미니멀한 색상 팔레트
- **폰트:** 기본 시스템 폰트
- **인터랙션:** 기본적인 드롭다운 메뉴, 텍스트 입력창, 버튼 사용

### 기능
- **상황 선택:** 사과, 요청, 보고, 거절 등 4가지 상황 선택 가능
- **톤 선택:** 정중하게, 부드럽게, 간결하게 등 3가지 톤 선택 가능
- **상황 설명:** 사용자가 구체적인 맥락을 텍스트로 입력
- **문구 생성:** 선택된 옵션에 따라 미리 정의된 더미 텍스트를 즉시 표시
- **복사하기:** 생성된 문구를 클립보드에 복사하는 기능

---

## 3. 현재 요청 계획: 사용자 유지율 증대 업데이트

### 목표
사용자가 서비스를 한 번 사용한 후에도 다시 방문하고 싶도록, 사이트의 효용성과 매력도를 높입니다.

### 세부 실행 계획

1.  **디자인 시스템 현대화 ("Bold" 원칙 적용):**
    *   **폰트:** 가독성과 미학을 고려하여 Google Fonts의 웹 폰트 (예: 'Pretendard')를 적용합니다.
    *   **색상:** 활기차고 현대적인 느낌을 주는 신규 색상 팔레트를 도입합니다.
    *   **레이아웃:** 여백과 간격을 재조정하여 시각적 안정감을 높입니다.
    *   **시각 효과:** 카드, 버튼 등 인터랙티브 요소에 부드러운 그림자(drop shadow)를 적용하여 입체감을 부여합니다.

2.  **사용자 경험(UX) 개선:**
    *   **로딩 상태 피드백:** 문구 생성 버튼 클릭 시, 결과가 나오기 전까지 로딩 스피너를 표시하여 시스템이 작동 중임을 알립니다. 이는 향후 실제 AI API 연동 시 필수적인 요소입니다.
    *   **생성 기록(History) 기능 추가:**
        *   사용자가 생성한 문구들을 최대 5개까지 목록 형태로 보여줍니다.
        *   이 기록은 브라우저의 `localStorage`에 저장되어, 사용자가 페이지를 새로고침하거나 재방문해도 유지됩니다.
        *   기록된 문구를 클릭하면 해당 내용이 다시 결과창에 표시되어 쉽게 재사용할 수 있습니다.

3.  **AI 기능 시뮬레이션 고도화:**
    *   `setTimeout`을 활용하여 실제 네트워크 지연(약 1초)을 시뮬레이션합니다.
    *   이를 통해 로딩 스피너의 작동을 자연스럽게 테스트하고, 사용자에게 더 현실적인 경험을 제공합니다.
